# community-backend 统一错误码字典

> 更新时间：2026-04-25  
> 代码定义：`community-backend/src/main/java/com/community/backend/common/error/ErrorCode.java`

## 1. 约定

- `code = 0`：成功
- `code != 0`：失败（按模块分段）
- 业务异常统一抛 `BizException`
- 参数校验异常由 `GlobalExceptionHandler` 统一转换为标准码

示例：

```json
{
  "code": 23001,
  "message": "帖子不存在或不可见",
  "data": null,
  "requestId": null
}
```

## 3. 编码规范（后续开发）

1. 新增业务异常时，先在 `ErrorCode` 增加枚举，再抛 `BizException.of(...)`。  
2. 不再直接 `throw new IllegalArgumentException(...)`。  
3. 前端联调只需关注 `code` 与 `message`，HTTP 状态码保持兼容。
